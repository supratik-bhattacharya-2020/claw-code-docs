# Terminal Replay Sandbox

Open `index.html` directly in a browser, or serve it:

```sh
cd sandbox
python -m http.server 8080
# → http://localhost:8080
```

## Controls

| Key       | Action          |
|-----------|-----------------|
| Space     | Play / Pause    |
| ← →       | Prev / Next frame |
| 1-4       | Speed (0.5x–4x) |

Click the progress bar to scrub. Switch scenarios via tabs.
